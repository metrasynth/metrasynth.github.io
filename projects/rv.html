Python interface to SunVox DLL¶

rv makes use of Python’s ctypes module to provide access to all of the SunVox functions described in the sunvox.h header file.

Python interface to SunVox file format¶

The .sunvox and .sunsynth file formats follow the IFF container format. Python’s chunk module handles reading and writing that format, and rv provides a SunVox file reader, a SunVox file writer [1], and a SunVox object model.

Note

The interpretation of SunVox file formats is based on “clean room” style inspection of what SunVox writes to disk when a file is edited a certain way.

There is currently no official documentation regarding the SunVox file format [2], but until such time there is, rv intends to act as a de facto source of documentation about the format.

Footnotes¶

[1]Work on the file writer has not yet begun.
[2]See the “File format .sunvox” thread.
